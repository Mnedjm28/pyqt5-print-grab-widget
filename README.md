## Description
This project is a **desktop application built with PyQt5** that demonstrates how to **print GUI widgets** directly from a PyQt5 application. Users can create text editors in the interface and print their content as it appears on the screen.

The script shows how to:
- Capture PyQt5 widgets as pixmaps  
- Print the captured widgets using `QPrinter`  
- Use `QPainter` to render the content for printing  

---

## Features
- Display multiple `QTextEdit` widgets with editable text  
- Print the entire application window or specific widgets  
- Capture widgets as images before printing  
- Easy-to-use GUI with a single print button  

---

## Installation
1. Clone the repository:  
    ```bash
    git clone https://github.com/your-username/pyqt-widget-printer.git
    ```
2. Navigate to the project folder:
    ```bash
    cd pyqt-widget-printer.
    ```

3. Install dependencies from the requirements.txt file:
    ```bash
    pip install -r requirements.txt
    ```
    
4. Run the application:
    ```bash
    python main.py
    ```
